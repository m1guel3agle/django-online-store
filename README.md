##Cómo ejecutar

pip install django
python manage.py migrate
python manage.py runserver

## Que hace el proyecto

Este proyecto es una aplicación básica desarrollada en Django donde se creó una estructura inicial siguiendo el patrón MVC. Desde el inicio se configuró el proyecto y una aplicación llamada pages, en la cual 
se implementaron vistas para una página principal Home y una página informativa About. Luego, se creó un modelo de productos conectado a una base de datos, permitiendo listar los productos y ver
el detalle de cada uno mediante rutas dinámicas. Ya a lo ultimo se habilitó el panel de administración de Django para gestionar los productos de forma visual, demostrando el uso de vistas, modelos, URLs y base
de datos dentro de un proyecto web funcional.
